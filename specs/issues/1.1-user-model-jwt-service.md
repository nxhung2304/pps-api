## Step-by-step Guide

**Files to create/modify:**

- `app/models/user.rb` — Model logic and validations.
- `app/services/jwt_service.rb` — JWT encoding/decoding logic.

- `test/services/jwt_service_test.rb` — Token lifecycle tests.

**Key decisions:**
- Use `Rails.application.secret_key_base` as the HMAC secret.
- Default token expiration to 24 hours.

**Non-obvious snippets**:
```ruby
# app/services/jwt_service.rb
class JwtService
  SECRET_KEY = Rails.application.secret_key_base

  def self.encode(payload, exp = 24.hours.from_now)
    payload[:exp] = exp.to_i
    JWT.encode(payload, SECRET_KEY)
  end

  def self.decode(token)
    body = JWT.decode(token, SECRET_KEY)[0]
    HashWithIndifferentAccess.new body
  rescue JWT::DecodeError
    nil
  end
end
```
